# 🔐 Secure Web Check

AI-powered web security testing tool built using FastAPI to detect basic vulnerabilities in web applications.

## 🚀 Features
- URL-based security scanning
- Basic vulnerability detection (XSS, SQLi)
- FastAPI backend
- JSON API responses

## 🛠️ Tech Stack
- Python
- FastAPI
- Uvicorn
- Requests

## ▶️ How to Run

```bash
git clone https://github.com/Itsdevilak/secure-web-check.git
cd secure-web-check
pip install -r requirements.txt
uvicorn app.main:app --reload
